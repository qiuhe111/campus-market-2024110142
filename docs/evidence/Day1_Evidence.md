## 学习收获

1. **Vue 3 项目启动流程**
   以前写 Vue 只是照搬模板，今天读 main.ts 才真正理解 `createApp` → `app.use()` → `app.mount('#app')` 每一行在做什么。Pinia 和 Router 都是通过插件机制注册，这个"插件体系"是 Vue 3 扩展能力的基础。

2. **SPA 路由工作原理**
   Router 的作用不只是配置路径，它通过 `<RouterView>` 动态渲染当前匹配的组件，实现无刷新页面切换。路由守卫则可以在导航前后执行逻辑（如登录校验）。

3. **目录结构设计的工程意义**
   按职责分层（views/components/stores/api）不是形式主义——当组件、状态、API 请求混在一起时项目会迅速变得难以维护。这种分层让每段代码都有"固定归属"，新成员也能快速定位。

4. **AI 协作的节奏**
   通过 Opencode 逐步分析目录、代码、技术栈，发现好的 Prompt 应该是"分步骤、明确上下文"的，而不是一次性扔一个大问题。
